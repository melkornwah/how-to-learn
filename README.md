# **Научиться учиться**

## **Описание**

Данный проект является не только визуализацией возможностей HTML и CSS, но ещё и инструкциями к улучшению обучаемости!


## **Функциональность**

В этом проекте используются одновременно grid и flex контейнеры. Это позволяет расширить кроссбраузерность!
Так же, в качестве ихображений, используются только необходимые фрагменты, которые пользователи захотят сохранить себе. Остальные же - фоновые!


## **Планы по доработке проекта**

* Переписать flex-таблицы на grid, для лучшей оптимизации и расширения кроссбраузерности!
* Подключить дополнительные и альтернативные стили для проекта через fonts для получения более приятного и органичного восприятия!
